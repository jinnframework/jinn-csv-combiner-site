CSV Combiner

A lightweight tool to merge CSVs with Python

Download CSV Combiner (Universal Python)

Requires Python 3.7+

# Run the app
$ pip install -r requirements.txt
$ python csv_combiner.py

✨ Features

  • Combine multiple CSV files into a single output
  • Automatic header management
  • Supports drag & drop (when run with GUI)
  • Cross-platform: works on Windows, macOS, and Linux
